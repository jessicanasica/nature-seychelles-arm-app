📲 How to install on Android

  1. Tap Download APK above — the APK file will start downloading.
  2. When the download finishes, tap Open or find the file in your notifications.
  3. Your phone may show a security warning — this is normal. Tap Settings, then turn on "Allow from this source", and go back.
  4. Tap Install and wait for it to finish.
  5. Tap Open to launch Cousin Island ARM!
✅ This is safe. Android shows this warning for all apps installed outside the Play Store. This app is developed by ARM/Nature Seychelles and is completely safe to install.

🍎 How to install on iOS

  1. Tap Download IPA or Get on App Store above.
  2. If using the App Store, tap Get and confirm with Face ID, Touch ID, or your Apple ID password.
  3. Wait for the app to download and install.
  4. Open Cousin Island ARM from your Home Screen!
✅ This app is developed by ARM/Nature Seychelles for field work on Cousin Island.
